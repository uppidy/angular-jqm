jqmButton (directive in module jqm )

Description

Creates a jquery mobile button on the given element.

If created on an anchor <a> tag, the button will be treated as a link button.

Usage

as attribute
<ANY jqm-button="{submit|reset}"
     [icon="{string}"]
     [iconpos="{left|right|top|bottom}"]
     [mini="{boolean}"]
     [inline="{boolean}"]
     [shadow="{boolean}"]
     [corners="{boolean}"]>
   ...
</ANY>

Parameters

  • jqmButton(optional) – {submit|reset=} –

    The button type - if specified, the button will be treated as an input with the given value as its type. Otherwise, the button will just be a normal button.

  • icon(optional) – {string=} –

    Defines an icon for the button

  • iconpos(optional) – {left|right|top|bottom=} –

    Defines the Position of the icon, default is 'left'

  • mini(optional) – {boolean=} –

    Whether or not to use the mini layout

  • inline(optional) – {boolean=} –

    Whether or not to make the button inline (smaller)

  • shadow(optional) – {boolean=} –

    Whether or not to give the button shadows (default true)

  • corners(optional) – {boolean=} –

    Whether or not to give the button shadows (default true)
