jqmFlip (directive in module jqm )

Description

Creates a jquery mobile flip switch on the given element.

Anything inside the jqm-flip tag will be a label.

Usage

as attribute
<ANY jqm-flip
     [ngModel="{expression}"]
     [disabled]
     [ngOnLabel="{string}"]
     [ngOnValue="{string}"]
     [ngOffLabel="{string}"]
     [ngOffValue="{string}"]>
   ...
</ANY>

Parameters

  • ngModel(optional) – {expression=} –

    Assignable angular expression to data-bind to.

  • disabled(optional) – {string=} –

    Whether this flip switch is disabled.

  • ngOnLabel(optional) – {string=} –

    The label which should be shown when fliped on.

  • ngOnValue(optional) – {string=} –

    The value to which the expression should be set when fliped on.

  • ngOffLabel(optional) – {string=} –

    The label which should be shown when fliped off.

  • ngOffValue(optional) – {string=} –

    The value to which the expression should be set when fliped off.
